# Projet de Simulation de Trafic Routier

Ce projet vise à simuler le trafic routier à l'aide d'un système de gestion intelligent des véhicules et des intersections.

## Objectifs :

Le projet a pour objectifs principaux :
- Simuler le comportement réaliste des véhicules dans un environnement routier.
- Gérer les intersections de manière efficace pour éviter les collisions et réguler le trafic.
- Fournir une interface utilisateur conviviale pour visualiser la simulation en temps réel.

## Installation :

Pour tester ou exécuter le projet, suivez ces étapes :

1. Assurez-vous d'avoir Python installé sur votre système.
2. Clonez le dépôt Git sur votre machine locale ou téléchargez uniquement le fichier Python
